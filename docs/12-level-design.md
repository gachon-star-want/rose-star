# Level Design

## 목표

최종 릴리즈는 44개 해넘이(스테이지)다(D059).

```text
4 달 x 11 해넘이 = 44 stages
44 stages x Easy/Hard = 88 practical challenges
```

달 = 막(Act) = 바오밥 주력 성장단계. 각 해넘이는 30~90초 안에 끝나는 마이크로 방어 퍼즐이다.

## 큰 원칙

- 각 해넘이는 하나의 아이디어를 가르친다.
- 같은 맵을 Easy/Hard가 공유한다.
- Hard Mode에서도 클리어 가능해야 한다.
- Easy Mode는 초당 별빛으로 실수 여지를 늘린다.
- 긴 웨이브보다 짧고 밀도 있는 변주를 우선한다.
- 새 타워/적/규칙은 한 번에 하나씩 소개한다.
- 모든 단계의 바오밥은 상시 등장하되, 그 달의 주력 단계만 변동한다(D052).

## 해넘이 데이터 구조

해넘이는 다음 요소로 구성한다.

| 요소 | 설명 |
| --- | --- |
| Header | ID, 달, 해넘이, 사용 가능 타워, 시작 별빛, 장미 꽃잎, Easy 수입 |
| Path commands | 적 경로(장미까지) |
| Build zones | 설치 가능 영역 |
| Waves | 적 타입, 수, 간격, 지연 |
| Flags | Hard 팔레트, 특수 규칙 |

맵 전체 타일을 저장하지 않는다. 경로와 설치 영역을 명령으로 만든다.

## Header 초안

```text
id              db  ; 0..43
moon            db  ; 1..4 (달)
sunset          db  ; 1..11 (해넘이)
tower_mask      db  ; 사용 가능 타워 bitmask
start_light     dw  ; 시작 별빛
petals          db  ; 장미 꽃잎(생명)
easy_income     db  ; Easy Mode 초당 별빛
kill_reward_mul db  ; 처치 보상 배율
path_count      db
build_count     db
wave_count      db
flags           db
```

## 명령 초안

```text
PATH  x y len dir
BUILD x y w h
BLOCK x y w h
WAVE  type count spacing delay
RULE  id value
```

방향 값:

| 값 | 방향 |
| --- | --- |
| 0 | Right |
| 1 | Down |
| 2 | Left |
| 3 | Up |

## 밸런스 기준값 (확정 초안)

AI 구현·밸런싱의 출발점이다(D063/D064). 해넘이별 웨이브 구성은 이 기준값 위에서 채운다.

### 생명·경제

| 항목 | 값 | 근거 |
| --- | --- | --- |
| 장미 꽃잎(생명) | 모든 해넘이 5장 고정 | D063 |
| leak 피해 | 🌳거목·🐍뱀 2장, 나머지 1장 | D063 / 11 적 표 |
| 시작 별빛 (달1/2/3/4) | 60 / 90 / 130 / 180 | D064 (해넘이별 ±10 튜닝) |
| Easy 초당 별빛 | +2 | D019/D020/D064 |
| Hard 초당 별빛 | 0 | D020/D064 |
| 처치 보상(별빛) | 🌱1·🌿2·🪴4·🌳7·🐍4·🎩4·💼0·🍷3 | 11 적 표 |
| 타워 판매 환급 | 구매가 60% | D018 |

### 타워 값 (10에서)

| 타워 | Cost | 1차 해금 |
| --- | ---: | --- |
| 🐑양 | 20 | 1-1 |
| 🌋화산 | 40 | 2-1 |
| 💡가로등 | 45 | 2-6 |
| 🏜️사막바람 | 35 | 3-6 |
| 🦊여우 | 50 | 4-1 |

### 페이싱 가드레일

- 모든 해넘이는 Hard(초당 별빛 0)에서도 클리어 가능해야 한다(D064 시작 별빛 + 처치보상만으로).
- 한 해넘이 30~90초. 웨이브는 적 타입 1~2종 중심(11 웨이브 조합 원칙).
- 달1은 🐑양 1종만으로 풀리도록(2-1에서 🌋화산 해금) — 11해넘이가 한 종으로 길지 않게 경로/병목(D047) 변주로 지루함 방지.
- 시작 별빛은 그 달 신규 타워를 1~2개 살 수 있는 선에서 ±10 조정.

## 실패 기준

- Hard Mode로 이론상 클리어 불가능
- Easy Mode가 너무 쉬워 타워 배치 의미가 없음
- 새 요소가 설명 없이 여러 개 동시에 등장
- 한 해넘이가 90초를 자주 넘김
- 특정 타워 하나만 계속 정답이 됨
