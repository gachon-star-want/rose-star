## 용량 해석

1.44MB는 일반적인 1.44 * 1024 * 1024가 아니라 3.5인치 HD 플로피 디스크의 실제 물리 용량인 **1,474,560바이트**다.

측정 기준은 다음처럼 잡는다.

- 압축 파일 크기가 아니라 압축 해제 후 전체 제출 폴더 크기
- 실행 파일, DLL, 데이터 파일, 폰트, 이미지, 사운드, 문서가 모두 포함될 수 있음
- 게임 내부 압축은 가능
- 최종 실행 가능한 전체 구성은 1,474,560바이트 이하

## 제출 기준 추천

개발 중 안전하게 잡는 내부 제출 기준은 다음과 같다.

- 최종 제출물 목표 상한: **1,400,000바이트 이하**
- 긴급 여유분: **74,560바이트 이상**
- 마감 시간 내부 기준: **2026-09-04 22:00 KST**
- 최종 검증은 제출 전 압축 해제 폴더 기준으로 수행

최종 릴리즈에서는 별도 원칙을 적용한다.

- 최종 제출물 목표 크기: **정확히 1,474,560바이트**
- `1,474,559바이트`도 실패로 간주한다.
- `1,474,561바이트`는 공모전 규칙 위반이므로 절대 실패다.
- 개발 중에는 안전선을 유지하고, 릴리즈 직전에 남는 바이트를 의미 있는 게임 데이터로 채워 정확히 맞춘다.
- 남는 공간을 채우기 위한 데이터는 가능하면 게임이 실제로 읽어야 한다.
- 의미 없는 패딩은 최후 수단이며, 사용하더라도 릴리즈 메타데이터/체크섬/숨겨진 컨텐츠 테이블로 해석 가능하게 설계한다.

## 공모전에서 어필할 포인트

- 실제 1.44MB 용량 제한을 기술적으로 정면 돌파
- 최종 제출물을 3.5인치 HD 플로피의 물리 용량인 1,474,560바이트에 정확히 맞춤
- 어셈블리어/무엔진 개발로 공모전 주제와 구현 방식 일치
- 원작 "해넘이 44번"을 44해넘이 캠페인으로 옮겨 1.44MB와 라임을 맞춤(D059) — 단 디지털/플로피 컨셉은 세계관에 넣지 않음(D049, 순수 어린왕자 우화)
- 작은 파일 안에서도 방어 루프, 44개 해넘이, 사운드를 모두 갖춤
