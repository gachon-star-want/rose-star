## ▶️ 실행 방법 (Windows)

1. 위에서 `.exe`를 받습니다.
2. **더블클릭**합니다.
3. *"Windows가 PC를 보호했습니다"* 창이 뜨면 → **추가 정보 → 실행**
   (서명 안 된 자작 실행 파일이라 뜨는 정상 경고입니다.)

설치·런타임·인터넷 전부 불필요. 그래픽·폰트·레벨·효과음이 모두 그 한 파일 안에 들어 있습니다.

## 🌙 게임

- **44해넘이 캠페인** = 4달 × 11 (달마다 자라는 바오밥: 싹 → 새싹 → 묘목 → 거목, 44번째 거목 보스)
- **타워 5종** 🐑양 · 🌋화산 · 💡가로등 · 🏜️사막바람 · 🦊여우
- **적 8종** 바오밥 4단계 + 🐍뱀 · 🎩허영쟁이 · 💼사업가 · 🍷술꾼
- **보너스 모드** 🏮점등인의 별(무한, 2-11 클리어 해금) · 🏜️사막의 신기루(절차 생성, 3-11 해금)
- 도입·막간·엔딩 서사, 쉬움/어려움, 기록 코드 이어하기, 공유 카드

지켜야 할 것은 별 위의 장미 한 송이. 바오밥이 닿으면 꽃잎이 진다.

## 🛠️ 소스에서 빌드

FASM(Linux ELF32) + Docker 래퍼로 macOS/Linux에서 Windows PE를 빌드합니다.

```sh
make release        # dist/rose-star.exe (정확히 1,474,560 B)
make harness-all    # 44해넘이 전부 Hard 클리어 헤드리스 검증
make harness-mech   # 전투 메커닉 10종 검증
make render-test    # 게임 화면을 PNG로 덤프(맥에서 눈으로 확인)
```

기술: x86 32-bit Windows PE, FASM 어셈블리, 256×192 내부 해상도(4배 확대 1024×768),
4bpp 인덱스 스프라이트, 12×12 비트맵 한글 폰트, winmm 절차 생성 효과음.

## ✅ 검증 상태

- 게임 로직·렌더·44해넘이 밸런스: **헤드리스(Docker i386) 검증 완료**
- 실제 Windows 창/입력/소리: **첫 실행 검증 대기** — macOS(arm64)에선 x86 PE를 못 돌려서, 위 exe로 실기기에서 한 번 확인이 필요합니다. 점검표는 [`docs/17-windows-runtime-check.md`](docs/17-windows-runtime-check.md).
